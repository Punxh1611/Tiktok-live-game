# 🛠️ คู่มือการเตรียมเครื่องและติดตั้ง (Setup Guide)

สำหรับสตรีมเมอร์ที่เพิ่งเริ่มต้น และยังไม่เคยติดตั้งโปรแกรมสำหรับรันโค้ดมาก่อน ให้ทำตามขั้นตอนนี้เพื่อติดตั้ง **ภาษาและโปรแกรมที่จำเป็น** ทั้งหมดครับ

## 1. โปรแกรมที่ต้องติดตั้ง (Prerequisites)
ระบบนี้จำเป็นต้องใช้โปรแกรม 2 ตัวหลักในการทำงาน:
1. **Node.js** (แกนหลักสำหรับรันเซิร์ฟเวอร์และระบบหลังบ้าน)
2. **Git** (สำหรับดาวน์โหลดโค้ดเวอร์ชันล่าสุดจาก GitHub)

### สำหรับผู้ใช้ Windows (วิธีที่ง่ายที่สุด)
คุณสามารถติดตั้งทุกอย่างได้ง่ายๆ ผ่าน Command Prompt หรือ PowerShell:
1. กดปุ่ม `Start` (โลโก้ Windows) พิมพ์คำว่า `cmd` แล้วกด Enter
2. ก๊อปปี้คำสั่งนี้ไปวางแล้วกด Enter เพื่อติดตั้ง Node.js และ Git:
   ```cmd
   winget install OpenJS.NodeJS && winget install Git.Git
   ```
3. ปิดหน้าจอ cmd แล้วเปิดใหม่เพื่อให้โปรแกรมจำค่าที่ติดตั้งไป

### สำหรับผู้ใช้ Mac
1. เปิดโปรแกรม `Terminal`
2. ก๊อปปี้คำสั่งนี้ไปวางเพื่อติดตั้ง Node.js และ Git ผ่าน Homebrew (หากยังไม่มี Homebrew ให้ติดตั้งก่อน):
   ```bash
   brew install node git
   ```

---

## 2. คำสั่งเดียวจบ! ดาวน์โหลดและรันโปรแกรมทันที
เมื่อคุณติดตั้ง Node.js และ Git เสร็จแล้ว (ตามข้อ 1) หรือถ้าคุณมีในเครื่องอยู่แล้ว 
ให้เปิด Terminal / Command Prompt ขึ้นมา แล้ว **ก๊อปปี้คำสั่งนี้ไปวาง กด Enter ครั้งเดียว** ระบบจะทำการดาวน์โหลดโค้ด ติดตั้งส่วนเสริม และเปิดโปรแกรมให้คุณทันที!

```bash
git clone https://github.com/Punxh1611/Tiktok-live-game.git && cd Tiktok-live-game && npm install && npm start
```

**คำสั่งนี้จะทำอะไรบ้าง?**
1. `git clone ...` : ดาวน์โหลดโค้ดทั้งหมดจาก GitHub มาไว้ในเครื่องคุณ
2. `cd Tiktok-live-game` : เข้าไปในโฟลเดอร์โปรเจกต์ที่เพิ่งโหลดมา
3. `npm install` : ติดตั้งเครื่องมือและส่วนเสริมที่ต้องใช้ทั้งหมด
4. `npm start` : เปิดการทำงานของระบบ (เมื่อขึ้นว่า Server is running... แปลว่าสำเร็จ)

หลังจากรันคำสั่งนี้เสร็จ คุณสามารถเปิดเบราว์เซอร์เข้าไปตั้งค่าที่ `http://localhost:3000/settings` ได้เลยครับ!
