# 🕒 TikTok Live Timer (เกมต่อเวลาไลฟ์ด้วยของขวัญ)

ระบบเสริมสำหรับสตรีมเมอร์ TikTok ที่ต้องการทำคอนเทนต์ **"ชาเลนจ์เอาชีวิตรอด"** หรือ **"ต่อเวลาไลฟ์เมื่อได้รับของขวัญ"** 
เมื่อคนดูส่งของขวัญมาให้ เวลาบนหน้าจอจะเพิ่มขึ้นโดยอัตโนมัติตามจำนวนเหรียญที่กำหนดไว้

---

## 🚀 วิธีการติดตั้งและการใช้งานเบื้องต้น

สำหรับสตรีมเมอร์ที่ต้องการรันระบบนี้ด้วยตัวเองบนคอมพิวเตอร์ที่ใช้สตรีม (Local/Single-user)

> 💡 **สำหรับมือใหม่!** หากคุณยังไม่มีโปรแกรมหรือติดตั้งไม่เป็นเลย  
> ให้อ่านคู่มือการเตรียมเครื่องแบบละเอียดยิบที่: **[setup.md](setup.md)** 

### คำสั่งด่วน (One-line Setup)
หากคุณมี Node.js และ Git อยู่แล้ว ก๊อปปี้คำสั่งนี้ไปรันใน Terminal ทีเดียวจบ!
```bash
git clone https://github.com/Punxh1611/Tiktok-live-game.git && cd Tiktok-live-game && npm install && npm start
```

### ขั้นตอนที่ 1: ติดตั้งโปรแกรมรัน (วิธีแยกทีละขั้นตอน)
1. ต้องมี **Node.js** ติดตั้งอยู่ในเครื่อง
2. เปิด Terminal (หรือ Command Prompt) ในโฟลเดอร์โปรเจกต์
3. พิมพ์คำสั่ง `npm install` เพื่อติดตั้งส่วนเสริมต่างๆ
4. พิมพ์คำสั่ง `npm start` (หรือ `node server.js`) เพื่อเปิดใช้งานเซิร์ฟเวอร์
   - ระบบจะทำงานที่ `http://localhost:3000`

### ขั้นตอนที่ 2: ตั้งค่าในหน้า Settings (สำหรับสตรีมเมอร์)
1. เปิดเบราว์เซอร์ไปที่ `http://localhost:3000/settings`
2. กรอก PIN เริ่มต้นคือ **`1234`**
3. ปรับการตั้งค่าตามต้องการ:
   - **Seconds per Coin**: 1 เหรียญให้เพิ่มเวลากี่วินาที
   - **Minimum Coins**: ตั้งเป็น 0 หรือ 1 เพื่อให้นับทุกของขวัญ (หรือปรับสูงขึ้นเพื่อกันสแปมของเล็ก)
   - **Start Time**: เวลาตั้งต้น (สามารถแก้ไขแล้วกด Save เพื่อให้เวลาเปลี่ยนบนจอทันที)
   - **Max Time Limit**: เพดานเวลาสูงสุด เพื่อไม่ให้เวลาทะลุเกินความจริง

### ขั้นตอนที่ 3: นำภาพขึ้นจอไลฟ์ (OBS Studio)
1. เปิดโปรแกรม OBS 
2. ในช่อง Sources กดเครื่องหมาย `+` แล้วเลือก **Browser**
3. ตั้งชื่อว่า "TikTok Timer" แล้วกด OK
4. ในช่อง **URL** ให้ใส่ `http://localhost:3000/overlay`
5. ปรับ Width / Height ได้ตามความเหมาะสม (เช่น 800 x 600)
6. **สำคัญ:** ตรวจสอบว่าพื้นหลังมันโปร่งใสและแสดงแค่นาฬิกา

## 🔗 วิธีเชื่อมต่อระบบเข้ากับ TikTok (ผ่าน TikFinity)

ระบบนี้จำเป็นต้องใช้เครื่องมือช่วยอ่านของขวัญจาก TikTok ที่ชื่อว่า **[TikFinity](https://tikfinity.zerody.one/)**

### 1. เปิดให้ระบบรับข้อมูลจากภายนอกได้ (Tunnel)
เนื่องจากเซิร์ฟเวอร์ของเรารันอยู่บนคอมพิวเตอร์ตัวเอง (localhost) TikFinity จึงมองไม่เห็น เราต้องเปิดอุโมงค์เชื่อมต่อ:
1. เปิด Terminal ใหม่ (ปล่อยหน้าต่างที่รัน `npm start` ไว้ห้ามปิด)
2. พิมพ์คำสั่ง `npm run tunnel` (หรือ `npx localtunnel --port 3000`)
3. ระบบจะสร้างลิงก์ขึ้นมาให้ เช่น `https://your-random-url.loca.lt` (ให้ก๊อปปี้ลิงก์นี้ไว้)

### 2. ตั้งค่า Webhook ใน TikFinity
1. ล็อคอินเข้าเว็บ [TikFinity](https://tikfinity.zerody.one/) แล้วเชื่อมต่อบัญชี TikTok ของคุณ
2. ไปที่เมนู **Actions & Events** (เมนูทางซ้ายมือ)
3. ดูที่ฝั่งซ้ายของจอตรงกล่อง **Actions** ให้กดปุ่ม **Create new Action**
4. ตรง Action Type ให้เลือกเป็น **Trigger Webhook**
5. เลื่อนลงมาตั้งค่าช่องต่างๆ ดังนี้:
   - **URL**: ให้ใส่ลิงก์ Tunnel ที่ก๊อปปี้มา ตามด้วย `/webhook/gift` (ตัวอย่าง: `https://your-random-url.loca.lt/webhook/gift`)
   - **Method**: เลือกเป็น **`POST`**
   - **Data Type**: เลือกเป็น **`JSON`** (หรือ `application/json`)
   - **Custom Body** (กล่องข้อความขนาดใหญ่): ให้ก๊อปปี้โค้ดด้านล่างนี้ไปวางทั้งหมดเลยครับ

   ```json
   {
     "eventId": "{event_id}",
     "giftName": "{gift_name}",
     "coins": "{coins}",
     "senderName": "{nickname}",
     "repeatEnd": {repeat_end}
   }
   ```
   *(หมายเหตุ: สามารถพิมพ์เองแล้วใช้ปุ่มตัวแปรที่ TikFinity มีให้กดแทรกได้เลย)*

6. กดบันทึก (Save Action) ด้านล่างสุด
7. ไปที่กล่อง **Events** (ฝั่งขวา) สร้าง Event ใหม่ตั้งชื่อว่า "Sending Gift"
8. เลือกเหตุการณ์เป็น **Any Gift** และกดเพิ่ม Action ที่เราเพิ่งสร้างมะกี้เข้าไป

### 3. ทดสอบการทำงาน
1. ลองเปิดไลฟ์ใน TikTok (สามารถตั้งเป็น Private Live ได้)
2. หรือใช้ปุ่ม "Test Event" ของ TikFinity (ถ้ามี)
3. หากระบบเชื่อมต่อถูกต้อง เมื่อมีของขวัญส่งเข้ามา เวลาบนหน้าจอ OBS จะเพิ่มขึ้น และมีชื่อคนส่งเด้งขึ้นมา!

---
*หมายเหตุ: ทุกครั้งที่ปิดคอมพิวเตอร์และกลับมาสตรีมใหม่ คุณจะต้องรัน `npm run tunnel` เพื่อเอาลิงก์ใหม่ ไปอัปเดตในช่อง URL ของ TikFinity เสมอ (ยกเว้นคุณจะใช้ Cloudflare Tunnels ที่สามารถทำให้ลิงก์คงที่ได้)*
